# XSA Budget App

## Descrição
O **XSA Budget App** é um aplicativo móvel desenvolvido para a empresa **XSA Uniformes**. Ele permite que os usuários solicitem orçamentos para confecção de uniformes personalizados, com opções de pagamento via PIX ou Cartão de Crédito. O aplicativo gerencia o fluxo do orçamento desde o preenchimento do formulário até a conclusão do pagamento.

## Funcionalidades
- Preenchimento de formulário de orçamento
- Geração e exibição do valor final do orçamento
- Métodos de pagamento: PIX e Cartão de Crédito
- Acompanhamento do status de pagamento
- Envio de notificações e confirmação de pagamento

## Tecnologias Utilizadas
- **React Native**: Para o desenvolvimento mobile.
- **React Navigation**: Para navegação entre telas.
- **Redux**: Para gerenciamento de estado.
- **AsyncStorage**: Para persistência local de dados temporários.
- **Axios**: Para comunicação com o backend.
- **Node.js e Express**: Para o backend (API RESTful).
- **MongoDB**: Para armazenar dados dos orçamentos.
